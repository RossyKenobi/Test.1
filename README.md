# F.Y. | Instagram Portfolio

A minimalist, high-performance photography portfolio built with Astro and integrated with Instagram carousel extraction.

## ✨ Features

- **Minimalist Design**: A clean, aesthetic interface focused on visual storytelling.
- **Instagram Integration**: Automated extraction of Instagram carousel posts to local high-resolution storage.
- **Premium Lightbox**: PhotoSwipe integration with dynamic aspect ratio handling (supports portrait/vertical shots without distortion).
- **Responsive Layout**: Fluid masonry grid that adapts from mobile to widescreen.
- **Fast Performance**: Built with Astro for optimized static site generation and zero-JS footprint where possible.

## 🛠️ Technology Stack

- **Framework**: [Astro](https://astro.build/)
- **Styling**: Vanilla CSS (Minimalist design system)
- **Gallery**: PhotoSwipe 5
- **Deployment**: GitHub Pages (via GitHub Actions)

## 📸 Local Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Dev Server**:
   ```bash
   npm run dev
   ```

3. **Build Product**:
   ```bash
   npm run build
   ```
